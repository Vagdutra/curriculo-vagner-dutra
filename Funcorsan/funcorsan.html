Análise de Dados Críticos

Desenvolvimento de lógica SQL complexa para auditoria de planos e integridade de contribuições previdenciárias.

query_auditoria_contribuicao.sql
SELECT p.nome_participante, pl.codigo_plano, pl.data_adesao, r.valor_contribuicao, -- Substitui DECODE por CASE para portabilidade CASE pl.status WHEN 'A' THEN 'Ativo' WHEN 'I' THEN 'Inativo' ELSE 'Desconhecido' END AS desc_status, r.dt_competencia AS competencia_reserva FROM tb_participante p INNER JOIN tb_plano pl ON p.id_participante = pl.id_participante INNER JOIN tb_reserva r ON pl.id_plano = r.id_plano AND r.dt_competencia = (SELECT MAX(dt_competencia) FROM tb_reserva WHERE id_plano = pl.id_plano) WHERE pl.status = 'A' AND (r.valor_contribuicao <= 0) ORDER BY pl.data_adesao DESC;
